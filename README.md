# Snake Game

A classic Snake game built with Godot 4.x.

## 项目状态

- **仓库:** LuckyJunjie/snake-game
- **引擎:** Godot 4.x
- **状态:** 开发中

## 游戏特性

- 经典贪食蛇玩法
- 计分系统
- 开始/结束界面
- 音效和特效
